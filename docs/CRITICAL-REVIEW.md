# Critical Review

This file tracks the main risks that still matter after the repo rebaseline.

## Active Risks

1. (Critical) The core letter prose still does not exist in sendable form.
- This is intentional, because Max wants to write the main letter copy himself.
- Risk: the project can feel structurally tidy while still not being close to a sendable message.
- Mitigation: keep the next work focused on the commented template, the section purposes, and draft evaluation rather than on polishing surrounding materials.

2. (Critical) The support package is not yet real enough to back the letter.
- `docs/index.html` and `docs/why.html` are now placeholders rather than finished public-facing material.
- Risk: the letter may eventually point toward resources or an explainer that are not ready, weak, or misaligned.
- Mitigation: treat the links page and explainer as part of the letter package, not as optional garnish.

3. (Major) The personalization strategy is still only partly operationalized.
- The MP research data exists, but the commented letter scaffold and the process for scaling dynamic MP-specific sections are not finished yet.
- Risk: either the letters stay too generic, or personalization gets rushed and starts straining weak connections.
- Mitigation: keep using reviewed examples, then scale carefully from those examples.

4. (Major) The project still needs a real send pipeline.
- Printing, paper, duplex layout, envelopes, addressing, freepost/postage rules, batching, and tracking are not yet locked down.
- Risk: the writing work gets ahead of the physical workflow and the last mile becomes a scramble.
- Mitigation: once the message is stable enough, move promptly into an operational pilot rather than treating sending as a final afterthought.

5. (Major) The strategy still needs validation against real readers.
- There is a clear intended tone and audience model, but no confirmed evidence yet that outside readers experience the letter that way.
- Risk: the draft could sound more self-important, more vague, or more alarmist than intended.
- Mitigation: test representative drafts with humans before scaling the full batch.
